# Olá! 👋 Eu sou o Julio Vitor Rufino

Sou Analista de Sistemas com foco em desenvolvimento web, banco de dados e integração de sistemas corporativos. Tenho experiência em criar soluções automatizadas, desenvolver dashboards de análise de dados e gerenciar infraestruturas técnicas.

## 🛠 Tecnologias e Habilidades

### 🗄️ Banco de Dados & ERP
*   **Banco de Dados:** Oracle DB, MySQL.
*   **Linguagens de Consulta:** SQL Avançado (criação de queries complexas, otimização e regras de negócio).
*   **Ecossistema TOTVS:** Gestão de ambiente TOTVS RM, criação de Fórmulas Visuais, automação de alertas.
*   **Fluig:** Desenvolvimento e integração de Datasets.

### 💻 Desenvolvimento Web & Backend
*   **Linguagens:** PHP, JavaScript.
*   **Servidores & Ferramentas:** WampServer.

### ⚙️ Infraestrutura, DevOps & Automação
*   **Containers:** Docker.
*   **Sistemas Operacionais:** Linux (experiência com Zorin OS e troubleshooting de pacotes/drivers).
*   **Redes:** Gerenciamento de DNS Dinâmico (No-IP).
*   **Mobile & Scripts:** Automação e troubleshooting de dispositivos Android (HyperOS) via comandos ADB, além da criação de macros para fluxos de trabalho e rotinas de rede.

## 🚀 O que eu faço
*   Desenvolvimento de dashboards de vendas (PHP, MySQL e JavaScript) para visualização e análise de dados de produtos.
*   Integração de sistemas e manipulação de dados em ambientes corporativos estruturados.
*   Automação de tarefas sistêmicas, desde o envio de alertas por e-mail até macros de acesso.
*   Manutenção e configuração de ambientes de desenvolvimento locais e conteinerizados.
